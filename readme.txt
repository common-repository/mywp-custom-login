== Description ==
The easiest way to customize your WordPress Login Screen.
With MyWP Custom Login, create your own Custom WordPress Login Page with just a few clicks!
Add your Logo and Customize your Title tag
Change Logo Width and Height
Define the Login Page Background Color
Put a Background Image
Customize completely your Login Form
Remove links from the WordPress Login Page
Delete the WordPress Text by default
Find all the settings to customize your WordPress Login Page in Settings > Custom Login & Dashboard.
